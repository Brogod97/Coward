    마이페이지 (메인, 내 정보 수정), 이용 약관, 개발자 찾기, 공모전 관리

- [김정윤](https://github.com/kim-jeongyoon)

     마이페이지 (내 정보, 공모전 관리, 프로필 수정), 우승자 심사 및 선정 페이지, 참가자 제출품 모달창, 맞춤 공모전 페이지
